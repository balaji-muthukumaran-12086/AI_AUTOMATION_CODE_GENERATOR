## 1. Global Security & Compliance Configuration

### 1.1 Authentication

| Parameter | Value |
|---|---|
| **Mechanism** | OAuth 2.0 Bearer Token / Zoho API Key |
| **Header** | `Authorization: Zoho-oauthtoken {token}` |
| **API Scope** | `sdpodapi` |
| **Session** | `authentication="required"` on all endpoints |
| **CORS** | `Access-Control-Allow-Origin: trusted\|oauthtrusted` |
| **Allowed Methods** | `POST, GET, PUT, DELETE, OPTIONS` |
| **Allowed Headers** | `Accept, Authorization` |

### 1.2 CORS Response Headers

```
Access-Control-Allow-Origin: trusted|oauthtrusted
Access-Control-Allow-Credentials: true
Vary: Origin
Access-Control-Allow-Methods: POST,GET,PUT,DELETE,OPTIONS
Access-Control-Allow-Headers: Accept,Authorization
```

### 1.3 File Upload Security

| Control | Policy |
|---|---|
| **Antivirus Scan** | `antivirus="true"` on all file uploads |
| **Metadata Removal** | `removeExif1, removeIptc` for image uploads |
| **Max Attachment Size** | Module-dependent (e.g., 51200 KB for images) |
| **Filename Validation** | Regex `document_name`, max-len=230 |
| **Content-Type** | Validated via `content-type-name` (e.g., `image`) |
