## 7. Multi-ID Grouping Convention

- When multiple manual test cases map to one automation method, comma-separate IDs:
  ```java
  @AutomaterScenario(id = "SDPOD_AUTO_REQ_LST_028,SDPOD_AUTO_REQ_LST_029", ...)
  ```
- Rules: same module prefix, genuinely validated within the same method, description covers combined coverage
- Added to `copilot-instructions.md`, `java-test-conventions.instructions.md`, and `test-generator.agent.md`
